### Ajouté
- Prompt système AIDEAL prêt à copier-coller pour ChatGPT, Claude, Open WebUI (`prompts/system-prompt-aideal.md`)
- 180 paires DPO réparties sur les 8 catégories (22-23 par catégorie)
  - genre-inclusion : 23 paires (écriture inclusive, charge mentale genrée, VSS, leadership féminin, féminisme décolonial…)
  - techno-solutionnisme : 22 paires (IA générative, fracture numérique, GAFAM, low-tech, cloud souverain…)
  - vision-economique : 23 paires (SCOP/SCIC, finance solidaire, communs, insertion, commande publique…)
  - validisme-accessibilite : 22 paires (FALC, RGAA, neurodiversité, handicap invisible, design universel…)
  - inegalites-nord-sud : 22 paires (volontourisme, dette, souveraineté alimentaire, extractivisme numérique…)
  - ecologie-sobriete : 23 paires (sobriété, agroécologie, greenwashing, ZAN, éco-anxiété…)
  - gouvernance-pouvoir-agir : 23 paires (sociocratie, éducation populaire, plaidoyer, intelligence collective…)
  - diversite-parcours : 22 paires (intersectionnalité, pair-aidance, racisme systémique, ruralité…)
- Section « Utiliser AIDEAL sans fine-tuning » dans le README
